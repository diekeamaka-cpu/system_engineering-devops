#!/bin/bash

Scripts to practice Linux file & directory permissions.

## Scripts

### 0. My Name Is Betty (0-iam_betty)
Switch user to betty.

### 1. Who Am I (1-who_am_i)
Print effective username.

### 2. Groups (2-groups)
Print all groups current user belongs to.

### 3. New Owner (3-new_owner)
Change owner of hello to betty.

### 4. Empty! (4-empty)
Create an empty file hello.

### 5. Execute (5-execute)
Add execute permission to owner of hello.

### 6. James Bond (6-James_Bond)
Set hello permissions: owner&group none, others all.

### 7. John Doe (7-John_Doe)
Set hello permissions to rwx r-x -wx.

### 8. Directories (8-James_Bond)
Add execute permission to all subdirectories.

### 9. More Directories (9-directory_permissions)
Create my_dir with 751 permissions.

### 10. Change Group (13-change_group)
Change group of hello to school.
