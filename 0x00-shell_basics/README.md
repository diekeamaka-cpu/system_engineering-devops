# 0x00-shell_basics

This repository contains Bash scripts to practice basic Linux shell commands.

## Scripts

### 0. Where Am I? (0-current_working_directory)
Prints the absolute path of the current working directory.

### 1. Whatâ€™s in There? (1-listit)
Lists the contents of the current directory.

### 2. There's no Place Like Home (2-bring_me_home)
Changes the working directory to the user's home directory without shell variables.

### 3. The Long Format (3-listfiles)
Lists directory contents in long format with permissions, ownership, size, and modification date.

### 4. Hidden Files (4-listmorefiles)
Displays all files, including hidden ones, in long format.
