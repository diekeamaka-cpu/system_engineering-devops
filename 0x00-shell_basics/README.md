# 0x00-shell_basics

This repository contains a collection of Bash scripts intended to facilitate the practice and understanding of fundamental Linux shell commands.

## Scripts

### 0. Where Am I? (0-current_working_directory)
Outputs the absolute path of the current working directory.

### 1. What’s in There? (1-listit)
Displays the contents of the current directory.

### 2. There's No Place Like Home (2-bring_me_home)
Changes the working directory to the user's home directory without relying on shell variables.

### 3. The Long Format (3-listfiles)
Lists directory contents in long format, including file permissions, ownership details, file sizes, and the most recent modification timestamps.

### 4. Hidden Files (4-listmorefiles)
Displays all files—both visible and hidden—in long format.
